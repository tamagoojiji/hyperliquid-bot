# rsi-vwap（RSI-VWAP）

| 項目 | 内容 |
|---|---|
| インジ/手法名 | RSI-VWAP（VWAP系オシレーター。緑/赤ゾーンで買い売り目線） |
| カテゴリ | オシレーター／押し目買い・底値検知 |
| 対象時間軸 | 30分足・**4時間足**（底値当てに強い） |
| 対象銘柄 | BTC（BTCUSD Perp / BYBIT） |
| 元ネタ | NeMO（2024/09/30〜10/27、第122回） |
| 併用 | EMA200(赤太線)/EMA30(黄)/EMA7(青) |

> 出典画像: `assets/rsi-vwap/01〜03`

---

## 基本的なVWAPの見方
- **① 緑ゾーン → 買い目線ゾーン**
  - 緑から白へ変わる → **VWAP上昇開始**
- **② 赤ゾーン → 売り目線ゾーン**
  - 赤から白へ変わる → **VWAP下落開始**

### 発砲条件・使い方
- **30分EMA200（赤太線）を突破していない時に発砲する**。
- **30分EMA200を突破していない時の買い増しシグナル**として使う。

---

## 買い法則：緑→赤→赤（30分足）
- **緑 ⇒ 赤 ⇒ 赤 の2回目の赤になった時が買い**。
- 上昇相場の規定は、次の**どちらか**:
  - ❶ **200EMA以上**とする
  - ❷ **暗黒時代が終わっている**

---

## 4時間足での底値当て（第122回 2024.10.27）
- **4時間足のVWAPが2024年（半減期の年）の底値をめちゃめちゃ当てていた**。
- **4時間足の緑1回目が出てから、7本目のローソクで買い**（大体丸1日後）。
  - 実例: ❸本目 1月23日(火)21時〜 最安値38500$ → 7本目 終値39933 で買い場。
- 将来も当たるとは限らないから**分散の指標に**。**過信してはならない**。

### RSI-VWAP（4時間足）の緑ゾーン定義
- 4時間足において、**緑の線（RSI-30%のところ）より価格帯が下にならないと緑ゾーンではない**。
- **緑の線に当たるところ（髭でも）で買えたら、上昇傾向になることがここ最近多い**（要観察）。

---

## パラメータ例（画像より）
- RSI-VWAP INDICATOR 設定例: `17, 19, 80` / `17, 20, 80`（4時間足は `17 20 80` / 30.0 80.0 20.0 帯）。
- 4時間足の分析では RSI Channel・BB・EMA・MA・MACD・Divergence Histogram も併用（画像03「RSI予測値」）。
