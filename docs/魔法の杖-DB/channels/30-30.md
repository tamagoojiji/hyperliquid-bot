# 30-30（30分足 EMA30 戦略）

| 項目 | 内容 |
|---|---|
| インジ/手法名 | 30-30 ＝ **30分足のEMA30本**を軸にした3本EMA配列＋押し目買い |
| カテゴリ | 移動平均線（EMAリボン）／トレンドフォロー・押し目買い |
| 対象時間軸 | **30分足** |
| 対象銘柄 | BTC（BTCUSD Perp / BYBIT） |
| 元ネタ | NeMO（2024/09/03、2024/10/20） |
| 併用 | RCI3lines、RSI-VWAP魔法 |

> 出典画像: `assets/30-30/01〜03`
> ※名称の由来：「30分足の30本EMA」を略して **30-30**（RSIの30ではない）。

---

## 基本設定（30分足の3本EMA）
| 色 | EMA本数 |
|---|---|
| 🔵 青 | EMA **7本** |
| 🟡 黄 | EMA **30本** ← 主役 |
| 🔴 赤 | EMA **200本** |

- **1本前🍓のEMAがシグナル（確定足）**。

---

## 発砲条件（買い）
1. 30分EMAが**下から、赤 ⇒ 黄 ⇒ 青 の順に並んだ後**に
   （＝下から順に 赤200・黄30・青7 ＝ 上昇配列）
2. **価格が上から下がってきた時に、黄色線（EMA30）で買いシグナル点灯**。

> 実例: **4月10日14時 28202$ 買いシグナル確定**（価格が黄色線まで押して反発）。

---

## 暗黒時代（2024/10/20 定義）
「暗黒時代」＝トレンドが下向きに転じている期間。

- **① 30-30 買い発砲後**、
- **② 30分EMAが 上から、赤 ⇒ 黄 ⇒ 青 の順に並んだら 暗黒時代スタート**。
  （＝上から 赤200・黄30・青7 ＝ 下降配列）
- その後に **30-30の売りシグナルが点灯**する。そしていずれ **30-30の買い発砲へ進行**。この繰り返し。
- 単純に「**30分足で200EMAより下**」で暗黒時代を定義する場合もあるが、**両者覚えておくべき**。

---

## サイクル（まとめ）
```
下から赤→黄→青（上昇配列）→ 価格が黄色線に押し目 → 買い発砲
        ↓
上から赤→黄→青（下降配列）→ 暗黒時代スタート → 売りシグナル
        ↓
        …いずれ再び買い発砲へ（繰り返し）
```

| 状態 | EMA配列(下→上) | 意味 |
|---|---|---|
| 上昇 | 赤 → 黄 → 青 | 黄色線タッチで買い |
| 下降（暗黒時代） | 青 → 黄 → 赤（＝上から赤→黄→青） | 売り／200EMA下でも定義可 |
