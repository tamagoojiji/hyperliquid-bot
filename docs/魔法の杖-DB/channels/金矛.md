## 金矛シグナル（本体）
**NEOさんの金矛シグナル ＝ 基準線（赤色）と 遅行線（黄色）のクロス**
- ① 基準線を遅行線が**下に抜ける**クロス：**売りシグナル**
- ② 基準線を遅行線が**上に抜ける**クロス：**買いシグナル**

### 金矛プルバック1回目とは
- 金矛シグナル（買い/売り）が出た後、価格が上昇（下落）し、**初めてローソクが基準線をクロス**（タッチではない）することが「プルバック1回目（買い/売り）」。
- それぞれ**基準線に買い（売り）指値を置くだけで良い**。

## 日足シグナルの傾向・使い方

### 日足買いシグナル
- **雲実破**（雲抜けリーチ）が日足買いシグナル。
- 基準線下落傾向からの**基準線上昇開始**：基準線（赤色）の方向が相場の方向 → 上向いたら上昇入り。
- 日足より先に**2〜3日足が買い転換**した → 強い買いの可能性。
- 価格が**基準線を超える** → 上昇相場の一歩前進。
- 「金矛日足売り」は**真近の買い**（＝逆張り妙味）。
- 魔法の杖の「でんじ遅行チャート」全てを、**黄色線（一目の遅行スパン）が上方突破**した時に強い上昇。

### 日足売り発砲の傾向
- **1週間前後までに底打ち**が圧倒的大多数。
- 多日安値パターンも多い。
- **半年近い上昇相場の後**の金矛売りは、全部**短期的な底打ちパターン**。

### 月足の「上抜けの法則」
- **月足 金矛シグナル発砲直後 ＋ 月足パラボリくんの上タッチ ＝ 上抜けの法則**。

## 金矛予測値（基準線の先読み）
```
基準線 =（過去26本間の最高値 ＋ 過去26本間の最安値）÷ 2
※過去26本間 = 直近のろうそく足を含めたろうそく足26本分
```
- 基準線の予測値は上式で表せるため、**高値・安値を更新した時は予想が大きくズレる可能性がある**という認識を持っておく。

## 「雲」サブツール（第121回）
> オシレーター型の雲インジ（白三角/×で売買、IOオシレーター併用）。

- **条件**: 売転換（買転換）＋ **200EMA(赤太線)よりも上にある状態**
- **使い方**:
  - **白三角でロング**して、**×で撤退**する
  - **小さい×で部分利確**する
  - このインジを作った人は**30分足**が良いかなと
  - **NEOさん的には3時間足が好き**（検証途中）
  - 長い足（日足など）は過去の大相場の検証だと、**小さい×の3〜4回目で部分利確**するといい感じ

## 補足：一目均衡表の基礎

### 5本の線の定義
| 線 | 定義 |
|---|---|
| ① 基準線 | （当日を含めた過去**26日間**の最高値＋最安値）÷2 |
| ② 転換線 | （当日を含めた過去**9日間**の最高値＋最安値）÷2 |
| ③ 先行スパン1 | ｛(転換値＋基準値)÷2｝を**26日先行**させて表示 |
| ④ 先行スパン2 | ｛(当日を含めた過去**52日間**の最高値＋最安値)÷2｝を**26日先行**させて表示 |
| ⑤ 遅行スパン | 当日の終値を**26日遅行**させて表示 |

- 相場の方向性だけでなく、**相場参加者の心理状況を一目で確認できる**。
- 主に4要素で構成されるが、心理状況把握に**最も重要なのが遅行線（遅行スパン）**。

### ① 遅行線
- 当日の終値を26日前に遅行（チャートの左側にずらして）表示させたライン。
- 約1ヶ月前の価格と現在の相場価格（終値）を比較する役割（実質的には25日前と比較）。
- 遅行線が相場価格よりも…（※以下、原文ママ。一般的な一目の遅行線解釈とは上下が逆に記述されている点に留意）
  - **上にある場合**：25日前に購入していた人は含み損、ショートは含み益 → **売り手が有利**
  - **下にある場合**：25日前に購入していた人は含み益、ショートは含み損 → **買い手が有利**

### ② 先行スパン
- 先行スパン1：転換線と基準線の平均値を26日先に表示させたライン。
- 先行スパン2：過去52日間の最高値と最安値の平均値を26日先に先行して表示させたライン。
- → 先行スパンは、現在の相場価格の水準を**遅行線の倍の期間 = 50日（実質25×2）先行**させたライン。

### ③ 雲（抵抗帯）
- 先行スパン1と先行スパン2の間の空間を「雲」とよぶ。
- 相場のサポート／レジスタンスとして機能する。遅行線もこの雲で押さえつけられたりする。
- ローソク足が雲＞：上値抵抗帯の役割／ローソク足が雲＜：下値抵抗帯の役割。
- **雲のねじれ**（2本の先行スパンが交差したところ）→ **トレンド転換 or 加速**を示唆。

### ④ 遅行線の使い方（三役）
**順張り（買い）**
- 遅行線がローソク足を上回る → 買いシグナル
- 転換線が基準線を上抜ける
- ローソク足が雲の上限を上抜ける
- → **三役好転：強い買いシグナル**

**逆張り（売り）**
- 遅行線がローソク足を下回る → 売りシグナル
- 転換線が基準線を下抜ける
- ローソク足が雲の下限を下抜ける
- → **三役逆転：強い売りシグナル**

### 三役好転／三役逆転の定義
| 三役好転（強い買い） | 三役逆転（強い売り） |
|---|---|
| 買① 転換線 ＞ 基準線 | 売① 転換線 ＜ 基準線 |
| 買② ローソク足 ＞ 雲 | 売② ローソク足 ＜ 雲 |
| 買③ 遅行線 ＞ ローソク足 | 売③ 遅行線 ＜ ローソク足 |

### 転換線と基準線
- 価格により近い方が**転換線**、少し離れたところにあるのが**基準線**。
- 価格の動きに合わせて変化が大きいのが転換線、少し緩やかに傾くのが基準線。
- 上昇トレンドでは上にあるのが転換線、下降トレンドでは上にあるのが基準線。
