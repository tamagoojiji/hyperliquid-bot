# 窓埋めゲーム（CMEギャップ埋め）

| 項目 | 内容 |
|---|---|
| インジ/手法名 | 窓埋めゲーム ＝ CMEの窓（ギャップ）埋めを狙う逆張りデイトレ |
| カテゴリ | 時間・ギャップ手法（デイトレ/スキャル） |
| 対象時間軸 | 平日窓＝分足〜10分足 / 週末窓＝10分〜1時間足 |
| 対象銘柄 | BTC（Bybit BTCUSD / BTCUSDT で観測） |
| 元ネタ | NeMO（2024/11/10、原型は2022/07 Twitter） |

> 出典画像: `assets/窓埋めゲーム/01〜03`

---

## 基本原理
- **CMEの価格と各取引所の価格は乖離（スプレッド幅）がある** → **自分の使っている取引所の価格で見る**。
- **CMEの終値が決定される時刻** → 火曜日〜土曜日の **AM5時**（**冬時間はAM6時**）。
- CMEの終値は「取引最終時刻の6時の価格」ではなく、**5時に決定する清算値**が終値となる ＝ **窓（基準価格）はここで決まる**。

### 基準価格＝5時の清算値（＝黄色線）
- **基準価格より高い値段 → ショート脳**（＝上にあるので窓埋めで下がる想定＝ショート目線）
- **基準価格より安い値段 → ロング脳**（＝下にあるので窓埋めで上がる想定＝ロング目線）

### 利食いのコツ
- 利食いは**基準価格の少し下（上）で指値**しておくのが良い。
  → **基準価格と同値の指値をしている人が多い**から（少し手前で約定させる）。

---

## 週末 窓埋めゲーム（例：2日間と2時間）
- **土曜日AM5時のCME終値が基準価格**（週末窓ゲーム スタート）。例: 7/16(土)5時 21148$（Bybit BTCUSD）。
- 黄色線＝基準値（例 21148.0）:
  - 価格が**基準値より上 → ショート目線**
  - 価格が**基準値より下 → ロング目線**
- **月曜AM7時に終了**（＝週末窓ゲーム終わり）。例: 7/18(月)7時 21007$ でギャップダウン窓発生。
- トレード例（基準値付近で逆張り）:
  - ①7/17 21333 ショート → 21183 利確（150$幅）
  - ②7/17 21555 ショート → 21405 利確（150$幅）
  - ③ 21244 利確（311$幅）
  - ④7/18 20888 ロング → 20988 利確（100$幅）

---

## 平日 窓埋めゲーム（例：2時間）
- **平日は5時〜7時の2時間**が対象（CME終値5時→再開の窓）。
- 例: 7/21(木)5時 23651$ 平日窓ゲームスタート → 7時 23600$ 終了（ギャップダウン窓発生）。
- 黄色線＝基準値（例 23651.0）:
  - 5時〜7時、価格が**基準値より上 → ショート目線** / **下 → ロング目線**。
- トレード例:
  - ⑤7/21 23011 ロング → 23211 利確（200$幅）
  - ⑥ 23611 利確（650$幅）
