| 群 | 色 | 線 |

## 基本判定：イワシ vs クジラ（トレンド）
- **イワシ ＜ クジラ（黄色＜青＋赤）→ 下落トレンド相場**
- **イワシ ≧ クジラ（黄色≧青＋赤）→ 上昇トレンド相場**

> とりあえずGMMAを見るなら **30分・1h・4h・日足** の4つだけで良い。この4つでイワシがクジラより下なら**確実に下落トレンド**（第1回追記）。

### 反転チャートの活用
- **下落トレンド相場の時にNEOさんは反転チャートを使う**（上下反転表示）。
- PCの場合：**Alt + I（アイ）** / スマホ：右の表示価格を長押し→スケール反転。
- ロングバイアスがかかりにくくなり、素直にチャート分析ができる。

## 買い発砲は2つある（第3回）
1. **黄色の短期線の一番下にある線**（足の期間はなんでもいい）が、**長期線（赤と青）の一番上にある線を超えた時**が最初の買いシグナル。
2. **パーフェクトオーダーが決まった時**（上から 短期→中期→長期 なら買い、上から 長期→中期→短期 なら売り）。

### 200MA・200EMA 条件（第121回追記）
- ❶ **200MA（オレンジ線）と200EMA（赤線）の両方を超えないと、イワシクジラが反転しない**ようなシグナルにしている。
- ❷ **パーフェクトオーダーに関しては200MA（オレンジ）と200EMA（赤）は関係ない**。
- ❸ 日足でイワシクジラの**売り転換の時が買い時**の可能性（検証した人がいる）。
- ❹ **日足パーフェクトオーダーとイワシクジラの買い転換が同時**だった時が買い場だった。

## パーフェクトオーダー（PO）
- 複数の移動平均線（短期・中期・長期）が**順序よく並ぶ状態**で、短期的にも長期的にも強いトレンドを示す。
- 全ての通貨ペアと時間足で使える。移動平均線はEMA、期間は 25・75・200 が主流。
- **エントリーポイントは2つ目のクロス**：POでは「短期＆中期」「中期＆長期」の2つのクロスが必ず生まれる → その後ローソク足が**実体で直近高値or安値を抜けるのを待つ**。
- 利確の目安：トレ転サインの反対のクロス（短期と中期のクロスを目安）。
- ※EMA＝直近に敏感で初動を取りやすいが騙し注意 / SMA＝オーソドックスで騙しにあいにくい。

### POパターン（4種）→ 完全PO
1. イワシだけのパーフェクトオーダー完成
2. クジラだけのパーフェクトオーダー完成
3. イワシ＆クジラ**同時**のパーフェクトオーダー完成
- ⇒ 🍓 **完全パーフェクトオーダー完成**
- POには赤と青がある：**赤＝下落場面でのPO、青＝上昇場面でのPO**。

## プルバック1回目
### 定義
- 上昇 or 下落トレンドになって、**最初にローソク足が赤太線(200EMA)にタッチすること**。
- プルバックは上昇(下落)相場では、**黄色が青よりも完全に上(下)にある状態**（一本でも青線に黄色線がかかってたらダメ）で、**太赤線にタッチ**しないとプルバック1回目にはならない。

### 判定基準（確定条件）
- プルバック1回目が確定したかどうか ＝ **1本前の太赤線(200EMA)の確定値に、当該足の値が到達したかどうか**を判定基準とする。
- 理由：当該足の200EMAはリアルタイムでは確定まで変動する。途中で発砲していないのに確定時点で「振り返って発砲していた」ように判定され、**嘘の発砲**が起こることがあるため（特に急騰・急落時）。

### プルバック3パターン
1. **イワシ＆クジラ逆転した直後のプルバック**：下落トレンド ＆ PO未完成のまま🍓プルバック1回目発砲。
2. **逆転直後プルバック発生＆PO完成後にプルバック**：PO未完成のまま🍓プルバック1回目発砲後にPOが完成 → PO完成後に再度プルバック1回目発砲。
3. **逆転直後プルバック発生なく＆PO完成後にプルバック**：上昇トレンド＆PO、未完成時のプルバック発砲がなく、その後PO完成 → PO完成後にプルバック1回目発砲。

### 発生順序（重要）
- **短い足から順番にプルバックが起こる** → 次にどの足を見ればいいかが分かる → NEOさんはこの順番を重要視していそう。

## チャートの見方（アイコンの意味）
- チャート上部の**絵文字🌎**：イワシクジラの**逆転**を表す。
- チャート下部の **PO**：ここからパーフェクトオーダー（赤＝下落場面のPO、青＝上昇場面のPO）。
- EMA上の **"S"**：ここでプルバック（下落場面のタッチ）／上昇相場のタッチなら **"L"**。

### 点燈アイコン一覧
| 表示 | 意味 |
|---|---|
| ▲ 🌎🌎 買い転換🍎 | イワシ＞クジラ 買い暗示 |
| ▼ 🌎 売り転換🍎 | イワシ＜クジラ 売り暗示 |
| →PO ▲ | 上昇パーフェクトオーダー完成（クジラ） |
| →PO ▼ | 下落パーフェクトオーダー完成（クジラ） |
| **S**（赤） | プルバック1回目 売りシグナル点燈（200EMA・太赤線） |
| **S**（橙） | プルバック1回目 売りシグナル点燈（200MA・オレンジ線） |
| **L**（青） | プルバック1回目 買いシグナル点燈（200EMA・太赤線） |
| **L**（緑） | プルバック1回目 買いシグナル点燈（200MA・オレンジ線） |

### 順番発砲アイコンの意味
- **△** ⇒ 上昇トレンド / **△△** ⇒ 上昇トレンド＋パーフェクトオーダー
- **▲** ⇒ 下落トレンド / **▲▲** ⇒ 下落トレンド＋パーフェクトオーダー
- **▲△** ⇒ レンジ、**PL**＝プルバック1回目 待機中、**PL🍓**＝プルバック1回目 接近中
- **△①23181買** ⇒ 1回目『買』シグナル点燈の価格 / **▲④24376売** ⇒ 1回目『売』シグナル点燈の価格
- **〇の数字** ＝ 太赤線のクロス回数
- 🍓🍓🍓 **イベント売買には向きません**。

### イワシクジラ関係性の表
- **▲▲**：下落トレンド → プルバックが起こったら**ショート**
- **△△**：上昇トレンド → プルバックが起こったら**ロング**
- **苺マーク🍓**：そろそろこの足でプルバック1回目が起きそう

## マルチタイムフレーム（MTF）表の例
各時間足ごとに「イワシ/クジラの大小・トレンド記号・価格・クロス回数」を並べて、プルバックの伝播を管理する（画像03）。例:
```
1分足   イワシ＜クジラ ▲▲ 28020 ②      30分足  イワシ＜クジラ ▲▲ 28140 ③
5分足   イワシ＜クジラ ▲▲ PL🍓          60分足  イワシ＜クジラ ▲▲ 28068 ④
10分足  イワシ＞クジラ △△ 27991 ①      2時間足 イワシ＞クジラ △△ 27362 ①
20分足  イワシ＞クジラ △△ 28015 ①      4時間足 イワシ＞クジラ △△ PL🍓
```
→ 短い足から順にプルバックが点いていくのを追う。
