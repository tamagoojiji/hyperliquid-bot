## 買いのセットアップ手順

### ① 下落トレンドの確認
- **下落トレンドであること**を確認（高値・安値の切り下げ）。

### ② 最初の反発でMACDがGC（重要ポイント）
- 最初の反発が起き、**MACDがシグナルをGCする（下から上へ）**。
- この**GC時、直近のMACD（高値）をブレイクしていること**を確認する（← 重要ポイント）。
  - 「直近MACD高値」を「ブレイク時MACD」が上抜けている状態でGC。

### ③ 再びDCするのを待つ（方向確認）
- その後、**再びMACDがシグナルまで落ちてきてDCする**のを待つ。
- この時、**シグナルが狙いたい方向**であることを確認する。
  - 今回は下降→上昇への転換を狙うので、**シグナルは上向き**であることを確認。
  - また、**MACDの傾きは下向き**であることを確認する。
  - （＝ MACDはDC・MACDは下向き・シグナルは上向き）

### ④ Antiセットアップはここで終了 → エントリー・決済ルール
リンダ氏が「ABCパターンのリトレースメントに似ている」と記述 → 他の人の検証により:
- ❶ **エントリーポイント＝GCが確定した終値でエントリー**
- ❷ **損切り＝直近安値の下**
- ❸ **利益確定＝ABCパターンより、CD＝AB / CD'＝1.272AB / CD'＝1.618AB の3パターンが多い**と言われる
- ❹ **フィボナッチエクステンションを活用**する

---

## ABCパターン（利益確定の基礎）
```
AB = CD
CD' = (1.272 or 1.618) × AB
```
- 形: A(安値) → B(高値) → C(安値) → D → D'（上昇）。
- CD（＝ABと同値幅）、CD'（＝1.272AB or 1.618AB）が利確ターゲット。

### フィボナッチの引き方
- **直近安値を基点として、ABラインをフィボナッチエクステンションで引く**。
- **エントリーラインまで基点を持ってくる**。
- 利食いライン＝ **1 / 1.272 / 1.618**（例: 1=232.81、1.272=249.14、1.618=269.91）。
- 主要レベル例: 0 / 0.236 / 0.382 / 0.5 / 0.618 / 0.786 / 1 / 1.272 / 1.618 / 2。
- リスク/リワードも確認（実例パネル: リスク/リワード比 0.75 など）。

---

## 実例（BTC 30分足・BINANCE / 2025-02-03）
- **AB**（最初の反発の上昇）→ 押し目 → **CD**（本命の上昇）。
- MACD: 前回MACD → 今回MACD → **GC**（直近MACDブレイク）→ **MACD下向き＋シグナル上向きでDC** → エントリー。
- フィボ利食い: 1(99,199.29) / 1.272(100,519.42) / 1.618(102,198.71)。
