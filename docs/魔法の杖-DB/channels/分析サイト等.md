## 用語解説（建玉・ポジション系）

### 未決済建玉（Open Interest / OI）
- まだ決済されていない取引の数（市場全体のポジション数＝保有契約数）。
- **増加 → 新規ポジション増＝市場の活動が活発**。
- **減少 → ポジション解消＝取引が縮小**。

### ネットロング / ネットショート / ネットデルタ
- **ネットロング**：買いポジションを持つトレーダーの総数。増加＝強気（価格上昇期待）が強まる。
- **ネットショート**：売りポジションを持つトレーダーの総数。増加＝弱気（価格下落期待）が強まる。
- **ネットデルタ**：ロングとショートの差。プラス＝ロング多、マイナス＝ショート多。

### トップトレーダー ロング/ショート
- **口座ベース**：トップトレーダーのうち、ロング/ショートを持つ口座数の比率。
- **ポジションベース**：トップトレーダーが保有するロング/ショートのポジション量（資金量）の比率。

### ロング/ショート比率（Long/Short Ratio）
- ロングとショートの比率。**1超＝ロング優勢、1未満＝ショート優勢**。

### 資金調達率（Funding Rate / FR）
- ロングとショートのバランスを取るために支払われる資金の割合。
- **プラス＝ロングがショートに支払う**、**マイナス＝ショートがロングに支払う** → ポジションの傾きが分かる。

## Coinglass サイトの見方（全7項目）

### ① Bitcoin returns (%) — https://www.coinglass.com/ja/today
- ビットコインの**月次リターン・年次リターン**などを表で表示可能（デイリー/週次/月次/四半期）。

### ② Orderbook — https://www.coinglass.com/ja/merge/BTC-USDT
- 主要取引所（**OKX・バイナンス・bybit・Houbi**）の板をまとめて表示。
- いずれかの取引所で仕掛けられたり、特定価格でサポートされたりした時に**リアルタイムで観察可能**。

### ③ option Deribit：Volume by strike price（価格別出来高）— https://www.coinglass.com/ja/options/Deribit
- Deribit（**オプションの9割を占める取引所**）の価格別出来高。
- **緑の山（コール）が高い価格帯**＝その価格以上で利益が出る人が多い＝それ以上になると予想している。
- **緑の山が右（高価格帯）にシフトするほど市場は強気目線**。
- **赤の山（プット）が高い価格帯**＝その価格以下で利益＝それ以下になると予想。
- バランスでコール多ければ市場は上目線と読める。
- コールオプションの例：原資産(日経平均)を半年後29,000円で買う権利を500円で購入 → 半年後30,000円なら`30000-29000-500=500円`利益、28,000円なら権利放棄でオプション料500円の損失。

### ④ option Deribit：Volume by Expiration（満期別出来高）
- 下の数字がオプションの期日。どの期日を多くの人が見ているか分かる。相場によってはあまり使えないことも。

### ⑤ Funding Rate（資金調達率）— https://www.coinglass.com/FundingRate
- **緑（マイナス寄り）＝弱気より、赤（プラス寄り）＝強気より**。緑が多い＝市場は弱気＝売りに傾いている。
- **FRが正：ロングからショートに支払われる／FRが負：ショートからロングに支払われる**。
- FRは**現物価格との乖離を是正**するために設定されている。
- FRは証拠金ではなく**レバポジの総額に対してかかる**。
  - 例）証拠金5万で100万円の取引（レバ20倍）でFR0.01%の場合 → `1,000,000 × (0.01÷100) = 100円`。
- **限月先物・Futures（先物）には設定されていない**。
- 発生時間は取引所ごと：**bybitはJSTで 1時・9時・17時**（オーダー時間からの経過ではなく、この時刻に発生）。
- bybitの落ち着いた時のBTCUSDT FRは **0.01%くらい**が多い。FR幅は **−0.375〜−0.001%、0.001〜+0.375%**（2024.8現在）。

### ⑥ Liquidation Map（清算マップ）— https://www.coinglass.com/pro/futures/LiquidationMap
- **赤の点線＝現在価格**。
- レバごとに線が分かれ、各線はその価格帯にストップ価格（損切り価格）が置かれていることを意味する。
- **右の価格帯＝ショートが焼かれる価格、左の価格帯＝ロングが焼かれる価格**。

### ⑦ Grayscale の監視 — https://www.coinglass.com/Grayscale
- **Investors GBTC**（https://www.coinglass.com/GbtcInvestor）：GrayscaleのBTC連動投資信託をどの企業が買っているか。
- **Bitcoin Treasuries**（https://www.coinglass.com/BitcoinTreasuries）：BTC保有を公言している企業の保有状況。
- **Grayscale現物購入量（売却量）グラフ**：BTCの1日のマイニング量である **900BTCを超えた買い**があると価格を押し上げる要因になりうる。
