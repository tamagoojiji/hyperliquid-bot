# パラボリくん

| 項目 | 内容 |
|---|---|
| インジ/手法名 | パラボリック SAR（通称「パラボリくん」）＋ ボリンジャーバンド + 200EMA |
| カテゴリ | トレンドフォロー／反転検知（順張り・押し目買い） |
| 対象時間軸 | メイン **4h**（調整時は 6h にMTF）、上位確認に日足・週足 |
| 対象銘柄 | BTC（BTCUSD Perp / BYBIT想定） |
| 元ネタ | NEOさん系のパラボリ＋ボリバン運用。TradingView(トレビュ)のパラボリ仕様前提 |

> 出典画像: `assets/パラボリくん/01〜06`（Discord「魔法の杖」チャンネルより）

---

## 概要
パラボリック SAR（点で表示される「パラボリくん」）へのタッチを反転シグナルとして使い、
ボリンジャーバンドの「バンドウォーク」と組み合わせて、上昇相場の押し目買い／天井売りを判断する手法。
パラボリを **黄色・赤色の2種類**（パラメータ違い）で表示し、タッチ回数を「カウンティング」して相場の強弱と売買タイミングを測る。

---

## 使うインジと設定値

### パラボリック SAR（2色表示）
| パラボリ | 初期値 | 最大値 | 加速因数 (AF) |
|---|---|---|---|
| 🟡 黄色 | 0.02 | 0.2 | 0.02 |
| 🔴 赤色 | 0.025 | 0.05 | 0.025 |

- カウンティングは**赤でも黄色でもどちらでも良い**。NEOさんは黄色がほとんどだが、たまに先に赤が当たるので2つ表示している（2024.7.23）。

### 計算式
```
SAR = 前日のSAR ＋ AF ×（EP − 前日のSAR）
```
- **AF（加速因数）**: 一般に初期値0.02・最大値0.2。ローソク足が1本進むごとに0.02ずつ増加。
- **EP（極大値）**: 現在の上昇トレンドの最高値、または下降トレンドの最安値。
- **AFを大きくする → ダマシが増加**。

### ボリンジャーバンド
- NEOさんのボリバン：**2σ**。

### 移動平均
- **200EMA（赤太線）** を1回目タッチ判定の基準に使用。

---

## コア概念

### バンドウォークの定義（4h）
- **1本前のボリバン値 ＜ 当該足終値** となっているもの。
- **髭ではなく実体で見る**。カウントはEMAを割れているところからでも良さそう。
- 取引所によってはカウント無しになる足もある（画像①の注記）。

### カウンティング（パラボリへのタッチ回数）
- 上昇相場では、バンドウォーク後にパラボリへ複数回タッチしていく。その回数を数える。
- **②のあとに再度バンドウォークしてもカウントはリセットされず継続する**。
- 3回目タッチが2回目より低い → 弱い上昇の可能性。③がどこまで逃げ切れるかが重要。
- ③の発砲（反転点）が②より下になると④の位置も下がり、あまり上がれない。
  → ③はタッチせず上がってくれると、③タッチ後の④の位置も上になるので強い。

### 幻（まぼろし）パラボリくん ＝ トレビュ特有の仕様
- リアルタイムのパラボリ値は**発砲すると消える**。
- 実際にはパラボリ値へタッチしていなくても**反転するのがトレビュ(TradingView)のパラボリ仕様**（他チャートソフト＝本来のパラボリとは値が異なる）。
- 予測値が「1本前の安値より上」の場合に幻パラボリが発生しやすい。
- **表示値とタッチ判定値の乖離**（重要）:
  - 画面表示値 = 内部パラボリ値・1本前安値・2本前安値を比較し、**一番低い値**を表示（パラボリが下にいる時）。
  - しかし**タッチ判定は内部で持つパラボリ値を使う**。
  - 例：内部パラボリ100／1本前安値95／2本前安値90 → 画面表示は90だが、判定は100。
    現在値が100を割った瞬間に、見た目90に触れていなくても内部でタッチ扱い＝反転する。
  - → **ピタリ賞（表示値ピッタリ）で指値しない**。当該足の表示は切り上げて書くようにしている。
- 用語:
  - **当該足** = 反転を確認した足。
  - **1本前** = 当該足の1本前の足。

## エントリー条件

### ロング（買い）
- **基本形（上昇相場）**: 4h足、+2σ以上の**バンドウォーク5本目以上**のあと、4h足で**黄・赤パラボリくんにタッチ**でロングシグナル。
- **強い上昇相場**: バンドウォークしなくても、パラボリくんタッチで買い。
- **カウンティング上**: ボリバン ヨコヨコ**下限**タッチ（or/&）**赤色パラボリタッチ**で買いシグナル。
- **1回目タッチ = 買い**（後述MVRV参照）。タッチは赤でも黄色でもOK。

#### 4h足「1回目タッチ」の条件
- **バンドウォーク5本以上** かつ 1回目タッチ（**1本前が理想**）が **200EMA（赤太線）を原則超えている**こと。
- 5本以下だと弱い可能性あり。

### ショート（売り）
- **カウンティング上**: バンドウォーク後、黄色 or 赤色パラボリに**4回タッチ**・ボリバン ヨコヨコ**上限**タッチで売りシグナル（ここでパラボリタッチした場合はカウント＋売りシグナル？）。
- **4回目タッチ = 売り**（MVRV参照）。

### 補足条件
- 実体（終値）陽線**二本超え** ＝ バンド2本超えの条件。

## 上位足・複合シグナル

### 日足の複合買いシグナル（発砲）
次の3条件が揃うと**買いシグナル発砲**:
1. 日足 パラボリくん
2. 日足 MACD **陰転**
3. 日足 **新安値 2本目**

### 2016年・2019年の 4hパラボリくん × MVRV2.0超え の関連性
**結論**
- パラボリくん**1回目は買い、4回目タッチは売り** → どちらの年も当てはまる。
- 2016は4回目がほぼピーク、2019は2回目タッチ後そのまま続伸がピーク。
- MVRV2.0超えが確定するまでに1回目タッチは済んでいた。

**注意点**
- 1回目を割るような深い押しが3回目で出る可能性あり（2019）。
- 4回目後のボリバンタッチまで待つと怖い思いをしそう（2019）。
- EMA大割れ・全部下向き・ボリバン割れ → 撤退？？
- パラボリくん：**上昇相場では、下方にあるパラボリは時間経過とともに必ず上昇する**。

## 調整時（押し目）の立ち回り
- 赤色パラボリくんにタッチするところまで見て買う。
- 視点を **4h → 6h** に移し、**MTF（マルチタイムフレーム分析）**で見ながら、焼かれない程度に買い増し。

## 注意点まとめ
- AFを大きくするとダマシが増える。
- トレビュのパラボリは「幻パラボリ」仕様。表示値とタッチ判定値が乖離するので、表示値ピッタリの指値は避ける。
- 週足はカウンティングしない。
- 深い押し（1回目割れ）が3回目で出ることがある。EMA大割れ＋全下向き＋ボリバン割れは撤退検討。
