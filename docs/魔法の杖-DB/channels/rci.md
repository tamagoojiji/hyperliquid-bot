# rci（RCI）

| 項目 | 内容 |
|---|---|
| インジ/手法名 | RCI（Rank Correlation Index／順位相関指数） |
| カテゴリ | オシレーター／買われすぎ・売られすぎ・売買タイミング |
| 対象時間軸 | 4時間足・日足 |
| 対象銘柄 | BTC |
| 元ネタ | NeMO（2024.10.22、第122回 2024/10/27） |

---

## 核心メモ（2024.10.22）
- **❶ 4時間足の「整う」とは → 信号機が買いになること**。
- **❷ RCIにおける上昇相場は、RCIが80%を超えてから上がり続ける相場**。

## 第122回（2024/10/27）
- **日足：赤線 −80% で買い**。

---

## RCIの基礎
- 過去の一定期間の**レートと日数にそれぞれ順位をつけ、2つの順位相関を計算**する。
- 買われすぎ・売られすぎの判断材料。
- **−100% 〜 100%** で推移する。
- **RCI 80%以上で買われすぎ、RCI −80%以下で売られすぎ**。
- 短期・中期・長期のRCIを使って売買タイミングを見ることも可能。
- （参考: NEOさんのチャートでは RCI3lines を使用。青/赤/黄の3本）

---

## まとめ
| 状況 | 解釈 |
|---|---|
| 4時間足 信号機が買い | 「整う」＝買い環境 |
| RCI 80%超えから上がり続ける | RCIの上昇相場 |
| 日足 赤線 −80% | 買い |
| RCI −80%以下 | 売られすぎ（買い目線） |
| RCI +80%以上 | 買われすぎ |
