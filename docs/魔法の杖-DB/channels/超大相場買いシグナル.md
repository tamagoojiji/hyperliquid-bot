# 超大相場買いシグナル

| 項目 | 内容 |
|---|---|
| インジ/手法名 | 「超大相場」を示す買いシグナルの**重なり（コンフルエンス）チェックリスト** |
| カテゴリ | 総合判断（各インジの集約） |
| 対象時間軸 | 30分〜2ヶ月足まで横断 |
| 対象銘柄 | BTC |
| 元ネタ | Yuppi（2024/11/13） |
| 位置づけ | このDBの各チャンネル（パラボリくん/金矛/GMMA/ADX/RCI/RSI/CVD/VWAP/MM等）を横断的に束ねる総まとめ |

> ※これは1つのインジではなく、**「超大相場」を示すシグナルを列挙したチェックリスト**。多くが同時に重なるほど信頼度が上がる、という使い方（[ph](ph.md) の「シグナルの重なりが大事」と同じ発想）。
> ※「金矢」は [金矛](金矛.md)（かなぼこ）と同一と思われる（原文の表記ゆれ）。

## 使い方の考え方
- 単一シグナルでは判断せず、**上記が複数重なったときに「超大相場」の可能性**として捉える。
- 時間軸は 30分足〜2ヶ月足まで広く、上位足（週足・月足・2ヶ月足）のシグナルほど大相場につながりやすい。
- 一部は「逆シグナル／真逆の買いシグナル」（21・30番）＝通常の意味と逆に読む点に注意。

> 原文ママの用語（「ワンクラングラ」「勝ち転線」「相玉」「RT」等）はそのまま記載。意味が確定したら追記する。
