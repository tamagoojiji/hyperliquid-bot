ピボット活用案

Hyperliquid自動売買Bot - ピボットポイントを使った戦略拡張

ピボットポイントとは？

前日の高値・安値・終値から「今日の基準価格」と「反転しやすい価格帯」を自動計算する指標です。多くのトレーダーが意識するため、実際にその価格帯で反発が起きやすい特徴があります。

R3 $86,000 ─── 強い抵抗線（ここまで来たら上がりすぎ）
R2 $85,500
R1 $85,000 ─── ここで売りが入りやすい

P $84,500 ─── 今日の基準価格（ここより上なら買い優勢）

S1 $84,000 ─── ここで買いが入りやすい
S2 $83,500
S3 $83,000 ─── 強い支持線（ここまで来たら下がりすぎ）

計算式: P = (高値 + 安値 + 終値) / 3 を基準に、R1〜R3, S1〜S3 を算出

更新頻度: 1日1回（前日の確定値から算出）→ 当日中は固定値として機能

1
ピボット + RSI 30-30 強化
フィルター型 逆張り強化 Bot適性◎
エントリー（買い）

既存RSI 30-30のシグナル + 価格がピボットS1/S2付近

エントリー（売り）

既存RSI 30-30のシグナル + 価格がピボットR1/R2付近

条件

両方が重なった時だけエントリー → 回数は減るが精度UP

RSI 30-30シグナル発生: BTC $84,020 で買いシグナル
ピボットS1: $84,000
→ S1付近でRSIシグナルが重なった → エントリー

RSI 30-30シグナル発生: BTC $84,300 で買いシグナル
ピボットS1: $84,000（$300離れている）
→ ピボット付近ではない → スキップ

メリット

  • 既存RSI 30-30にフィルター追加するだけ
  • 開発工数が最小（数十行の追加）
  • 勝率が上がる（2重確認）

デメリット

  • エントリー回数が減る
  • ピボットとRSIが同時に重なる場面は限定的
  • 独立した新戦略ではない

利確・損切り

損切り: 次のピボットライン（S1で買い → S2が損切り）

利確: ピボットP（基準価格）or R1

RR: S1→P の幅で自動決定。通常1.5〜2.5程度

2
ピボットバウンス
逆張り Bot適性◎
エントリー（買い）

価格がS1/S2に到達 + 反転ローソク足確認 → ロング

エントリー（売り）

価格がR1/R2に到達 + 反転ローソク足確認 → ショート

ピボットライン: S1=$84,000 / P=$84,500 / R1=$85,000

14:00 BTC $84,050 → S1付近に到達
14:05 陽線確認（反転の兆候）
14:05 → $84,050 でロングエントリー
損切り: $83,500（S2）
利確: $84,500（P）→ +$450（RR ≒ 0.8）

より良いRR: S2付近でエントリーすれば
損切り: $83,000（S3）/ 利確: $84,000（S1）→ RR = 2.0

メリット

  • ルールが明確（ライン到達+反転で入る）
  • 損切り・利確が自動で決まる（ライン間の幅）
  • 取引頻度が十分（1日5〜10回）
  • レンジ相場で特に強い

デメリット

  • 強いトレンドではS1/R1を突き抜けて損切り連発
  • ライン間の幅が狭い日は利幅が小さい
  • 24h市場の「前日」区切りの設定が必要

利確・損切り

損切り: 次のピボットライン（S1買い → S2、R1売り → R2）

利確: 手前のピボットライン（S1買い → P、R1売り → P）

特徴: ライン間の幅 = 利幅。RRはライン配置で自動決定。S2/R2の方がRRが良い

3
ピボットブレイクアウト
ブレイクアウト 順張り Bot適性○
エントリー（買い）

価格がR1を上抜け + 出来高増加 → ロング（ブレイクに乗る）

エントリー（売り）

価格がS1を下抜け + 出来高増加 → ショート

ピボットライン: S1=$84,000 / P=$84,500 / R1=$85,000 / R2=$85,500

15:00 BTC $84,950 → R1に接近
15:05 BTC $85,020 → R1を上抜け + 出来高急増
15:05 → $85,020 でロングエントリー
損切り: $85,000（R1 = 抜けたライン）→ -$20
利確: $85,500（R2）→ +$480（RR ≒ 24）

メリット

  • トレンドに乗るので利幅が大きくなりやすい
  • 損切りが非常にタイト（抜けたラインの直下）
  • RRが非常に良い（損切り小 vs 利幅大）

デメリット

  • ダマシ（一瞬抜けて戻る）が多い
  • 出来高フィルターがないと勝率が低い
  • レンジ相場ではほぼ機能しない

利確・損切り

損切り: 抜けたピボットライン（R1ブレイク → R1が損切り）

利確: 次のライン（R1ブレイク → R2）

特徴: 損切りがタイトなのでRRが非常に良い。ただしダマシで勝率は40〜50%程度

4
ピボット + ボリバン ダブルフィルター
高精度 逆張り Bot適性○
エントリー（買い）

ピボットS1付近 + ボリバン下バンド付近 + 反転確認

エントリー（売り）

ピボットR1付近 + ボリバン上バンド付近 + 反転確認

条件

2つのサポート/レジスタンスが重なるゾーンでのみエントリー

ピボットS1: $84,000
ボリバン下バンド: $84,050
→ 2つのサポートが$50以内で重なっている → 強い反発ゾーン

ピボットS1: $84,000
ボリバン下バンド: $83,200
→ $800離れている → 重なっていない → スキップ

メリット

  • 2つの指標が重なる = 反発確率が高い
  • 勝率60〜70%が期待できる
  • 根拠が明確で判断に迷わない

デメリット

  • 2つが重なる場面は1日1〜2回程度
  • 取引頻度が低い
  • ボリバンが広がっている時はS1と重ならない

利確・損切り

損切り: S2 or ボリバン下バンドの外側（広い方を採用）

利確: ピボットP or ボリバン中心線（SMA20）

特徴: 損切りポイントが2つの指標で補強される。「ここを抜けたら本当にダメ」が明確

5
ピボット + VWAP ゾーントレード
ゾーン判定 Bot適性◎
ゾーン判定

価格 > P かつ > VWAP → 買いゾーン（ロングのみ）

価格 < P かつ < VWAP → 売りゾーン（ショートのみ）

P と VWAP の間 → 中立（エントリーしない）

エントリー

買いゾーンで → S1やVWAPまで押したら買い

エントリー

売りゾーンで → R1やVWAPまで戻したら売り

ピボットP: $84,500 / VWAP: $84,600
現在価格: $84,800
→ P($84,500)より上 かつ VWAP($84,600)より上 → 買いゾーン

14:30 BTC $84,620 → VWAPまで押した
→ 買いゾーン + VWAP到達 → ロングエントリー
損切り: P($84,500)を割ったら → ゾーンが崩れた
利確: R1($85,000)

メリット

  • まず方向を決めてからエントリー → 逆張り事故が少ない
  • ピボットとVWAPの2つで方向確認 → 信頼性が高い
  • 取引頻度が十分（1日3〜8回）
  • 「買いゾーンでは買いだけ」のルールが明快

デメリット

  • VWAPの計算に出来高データが必要
  • 中立ゾーン（PとVWAPの間）ではエントリーできない
  • 急なトレンド転換時にゾーン判定が遅れる

利確・損切り

損切り: ゾーンが崩れたら（P or VWAP を割ったら即撤退）

利確: R1/R2（買いゾーン時）or S1/S2（売りゾーン時）

特徴: 「方向が合ってる間は保有、方向が崩れたら撤退」のシンプルルール
